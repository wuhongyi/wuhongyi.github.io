Scientific Linux 安装 QStarDict

  • Linux
  • QStarDict
  • Software

       windows下我们知道很多软件可以划词翻译，这在linux下也不是问题。这里我们介绍QStarDict，之所以选择使用它是因为它安装是最简便的。

安装

        如果你想使用能够发音的，那你就先安装以下相关库：
在添加/删除中搜索以下

        如果你不需要发音，那就直接跳过以上步骤，在添加/删除 中搜索 stardict，将三个搜索结果都安装上，这样就安装好了。

下载我的词库

点击下载 我网盘的简易词库，基本够用了。

配置词库

当然，到这里只是安装了一个软件，没有词库！
在两个地方可以放置词库：

        将我给的 dic 文件夹 中的两个词汇库复制到以上任一位置，这就是添加词库的方式。
        添加词库之后还得设置词库，打开软件图形界面，左上角 设置-设置QStarDict-字典，这时候只要你的词库位置放置正确，在字典栏里就能看到你加入的词库，选择你所需要的词库，可以调整词库优先顺序。

软件的使用

        在 应用程序-附件-QStarDict 中打开软件，这时候在电脑右上方的通知栏会有一个绿色的图标，中间一个白色字母“D”。说明软件在后台运行着，可以取词翻译。
